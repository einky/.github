# .github

Organization-level GitHub configuration for the [**einky**](https://github.com/einky)
project.

## What lives here

| Path | Purpose |
|---|---|
| [`profile/README.md`](./profile/README.md) | The org landing page rendered at [github.com/einky](https://github.com/einky) |

> **Note.** GitHub renders `profile/README.md` (not this file) on the
> organization's public profile. This top-level README only documents the repo
> itself. Files placed here — issue/PR templates, `CODEOWNERS`, community-health
> defaults, reusable workflows — apply org-wide to any einky repo that doesn't
> define its own.
